# Домашнее задание №1: Анализ и портирование Z80-μLM

## Структура

- `task.md` — формулировка задания
- `report.md` — основной отчёт с анализом
- `experiments/` — код экспериментов (будет добавлено)

## Краткое описание

Анализ модели Z80-μLM (2-bit quantized language model для 8-bit процессора) и исследование возможности портирования на Arduino UNO/LilyPad с 2 KB SRAM.

### Основные выводы

1. Оригинальная модель (144K параметров, 36 KB) не влезает в Arduino SRAM
2. Возможна радикальная адаптация до ~3K параметров
3. Trade-off: 4x выше скорость vs ~95% потеря качества
4. Подход (2-bit QAT) переносим, архитектура — нет

## План экспериментов

1. **Z80 baseline**: запуск в эмуляторе, измерение latency
2. **Arduino UNO**: micro-модель с PROGMEM, proof-of-concept
3. **LilyPad @ 8MHz**: оптимизация под батарейное питание
